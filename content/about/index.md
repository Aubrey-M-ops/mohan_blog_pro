## 💻 Working Experience

#### TikTok (2022.5 - 2022.11)

> `TikTok` is one of the world's most popular apps, with over 1 billion active users globally each month.

- Contributed **60,000+ lines of code in 5 months**, developing core modules for the **Live-Streaming Workbench** and micro-frontend apps in the **One-stop Operations Platform**, driving product penetration to **90%**.  
- Optimized performance with **react-window** (−300 ms initial render) and **Recoil**, improving state management and app responsiveness.  
- Built **business components built on top of base components**, adopted **300+ times** across teams, reducing on-call issues by **60%**.  
 



#### Meituan (2022.11-2023.2) (2023.7-2024.7)

> Meituan is `China’s largest local services platform`, with over `400 million` monthly active users, effectively dominating the food delivery, hotel booking, and local services market.


- Built a **cross-platform UI library** with `41` base/domain components; 
  - Dynamic **PC/Mobile loading** cut shared code by **48%** 
  - Encapsulate **a unified glue layer** for web and mobile.
  - Saved an average of **~2.9 dev-days per feature**.

- **Led** front-end development for the **food-delivery core module** and **school-cafeteria vertical**, delivering **weekly feature updates** that drove **double-digit growth** in order volume.

- Built a **Babel-based AST tool** to map component usage and integrated it into **CI/CD**, reducing **code review time by 30%** and catching **dependency issues pre-merge**.
